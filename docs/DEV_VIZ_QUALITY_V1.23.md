# DEV 开发规范：可视化质量型优化 v1.23（方案草案，先评审后实施）

> 适用范围：Sonus 播放器全可视化系统（engine.js / Visualizer.jsx / Visualizer3D.jsx / index.css）。
> 文档性质：方案草案（先评审后实施）。
> 触发原因：v1.21 堆幅度过曝 → v1.22 已全部回调到合理基线。现在要在**不堆幅度**的前提下，让可视化"更精致、更有层次、过渡更顺"。
> 关联历史：`DEV_VISUALIZER_OPT_V2.md`(v1.21 范围放大) / `DEV_VIZ_TUNE_DOWN_v1.22.md`(v1.22 回调)。

---

## 0. 设计原则（本次关键技术取向）

v1.21/1.22 都在调**幅度（amplitude）**。本次转向调**质感（quality）**：

| 维度 | 做法 | 不做什么 |
|---|---|---|
| 色彩 | 更细腻的渐变、明度过渡、按频段分层着色 | 不靠提亮/增益 |
| 层次 | 频谱分段更细、边缘羽化、景深虚化 | 不靠放大半径 |
| 过渡 | 模式切换/播放-待机 更顺滑的缓动 | 不靠弹性过冲 |
| 待机 | 更有"呼吸感"的 idle 动画，而非静止 | 不靠加大振幅 |

**目标**：在 v1.22 的克制幅度上，让可视化"看起来贵"，而不是"看起来响"。

---

## 1. 域 A — 色彩与渐变升级（所有可视化共用）

### 1.1 2D 波（Visualizer.jsx）
- **径向频谱分层着色**：当前 `FILL_RINGS=12` 用单层 hsl 偏移。改为按 `layerProgress` 做**双色混合**（accent → 互补色/邻近色），峰部偏亮、谷部偏暗，层次更立体。
- **外圈光晕改为径向羽化**：`outerGrad` 增加 2–3 个中间 stop，边缘更柔（无硬边）。
- **频谱尖刺加渐变描边**：从根部 accent 到尖端透明，避免"硬线"。

### 1.2 3D 地形（Visualizer3D.jsx）
- **高度色带细化**：当前 `normH` 线性映射 accent。改为 3–4 段色带（谷底深蓝紫 → 山腰 accent → 雪顶近白），用 smoothstep 过渡，避免色带断层。
- **峰顶辉光改为按"坡度"而非仅 normH**：陡坡更亮（像受光面），增强立体感。
- **雾气随距离指数衰减**：远处更融于背景，强化景深。

### 1.3 星河（Visualizer3D.jsx galaxy 分支）
- 核心→外圈色温渐变更明显（炽热内核 → 冷调外晕），已有基础，增强 `coreMix` 曲线非线性。

---

## 2. 域 B — 频谱分段与边缘处理

### 2.1 2D 波
- `NUM_BARS=64` → 保持；但 `getSpectrumBars` 的 `logMin/logMax` 映射微调，让**低频段占更多 bar**（人耳/音乐低频信息更丰富），高频段压缩（高频本就稀疏）。
- 镜像频谱柱 `roundRectFill` 圆角加大 + 顶/底加 1px 羽化，消除锯齿感。

### 2.2 3D 地形
- 频率→高度映射 `freqWeight=(1-rN)` 改为 `Math.pow(1-rN, 1.3)`，让中心低频主峰更突出、外圈高频更收敛（视觉重心稳）。

---

## 3. 域 C — 过渡与待机动画

### 3.1 模式切换过渡
- 当前 3D 已有 `morphT` 跨模式形变。2D→3D / 3D→2D 切换时**加 150ms 淡入淡出**（`opacity` 缓动），避免硬切。
- 横向滑动切模式（v1.21 已实现）切换时，给当前可视化一个**快速淡出 + 新可视化淡入**。

### 3.2 播放 ↔ 待机过渡
- 地形 `terrainRise` 已有缓动。待机→播放时增加 **200ms 缓动起步**（不是瞬间 1.0），更自然。
- 2D 待机波形：当前是固定正弦。改为**随 accent 色相缓慢漂移的极慢呼吸**（周期 ~8s），待机也有"活着"的感觉但不抢戏。

### 3.3 鼓点响应质感（阈值已在 v1.22 调高）
- 鼓点脉冲 `beatPulseRef` 衰减 `BEAT_PULSE_DECAY=0.93` → `0.90`（脉冲更"脆"、收得更快，不拖泥带水）。

---

## 4. 域 D — 微交互质感增强（复用 v1.21 基础设施）

### 4.1 触摸涟漪（2D 已实现）
- 涟漪颜色随**落点处频谱能量**变化（能量高→更亮），而不固定 accent。
- 涟漪消散曲线改为 ease-out（先快后慢），更自然。

### 4.2 模式指示点（v1.21 已实现）
- 当前是纯色圆点。改为**当前模式点带呼吸光晕**（box-shadow 脉冲），其余点静态。

### 4.3 浮层（v1.21 下拉关闭已实现）
- 下拉关闭时增加**轻微透明度和模糊变化**（dragY 越大越透明），手势反馈更明确。

---

## 5. 实施顺序

1. 域 A（色彩/渐变）—— 视觉提升最直接
2. 域 B（分段/边缘）—— 细节精致度
3. 域 C（过渡/待机）—— 流畅度
4. 域 D（微交互质感）—— 点睛
5. `vite build` → commit → push `origin/main`

---

## 6. 验收标准

- [ ] 2D 波：有清晰明→暗层次，外圈无硬边，尖刺有渐变
- [ ] 3D 地形：谷底→山腰→雪顶有自然色带过渡，陡坡更亮，远处有雾融
- [ ] 模式切换有淡入淡出，不硬切
- [ ] 待机时可视化"活着"（极慢呼吸），但不抢戏
- [ ] 鼓点脉冲更脆、收得快
- [ ] `vite build` 通过，无报错

---

## 7. 风险控制

- 本次**全部是着色/过渡/缓动改动**，不碰幅度常量（v1.22 已定稿），不会再次过曝
- 若某处着色过重，单独回调该处系数即可，不影响全局
- 性能：新增的渐变 stop / 色带计算均为 O(像素/粒子) 常数级，无循环嵌套，开销可忽略
